# End-to-End Learning Project on ANN (TensorFlow + Keras)

## Overview
This project demonstrates an **end-to-end workflow** for building an **Artificial Neural Network (ANN)** using **TensorFlow with Keras** to solve the problem of **customer churn prediction**.  
Churn modeling helps businesses identify customers likely to leave, enabling proactive retention strategies.

---

## Features
- **Data preprocessing**: Encoding categorical variables, scaling numerical features.  
- **ANN architecture**: Sequential model with multiple dense layers and dropout for regularization.  
- **Training**: Backpropagation with Adam optimizer.  
- **Evaluation**: Accuracy, precision, recall, F1-score, ROC-AUC.  
- **Visualization**: Training/validation accuracy and loss curves, confusion matrix.  

---

## 🛠️ Tech Stack
- Python  
- TensorFlow & Keras  
- Pandas & NumPy  
- Matplotlib & Seaborn
